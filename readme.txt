This plugin allows you to create and display testimonials on multiple Styles.

== Description ==

Based on fully responsive & flexible jQuery carousel plugin help you to manage and display all testimonials, reviews or quotes on slider in multiple Styles!

= Plugin Features =

* Grid or Slider Layouts
* Translation Ready
* Fully Responsive & Compatible with all Major Browsers
* Easy to Use
* Custom Post Type
* Ability to add Designation to the Testimonials
* Ability To Manage and customize Testimonial Style Through Shortcode Builder
* Awesome Suppor

= How to Use =

1. For creating a new testimonial, you will simply need to click on the link "Add New"? under Testimonials Showcase from WordPress main menu.
2. You will be redirected to the new testimonial page where you can add the testimonial details in input fields.
3. Once the testimonials created, you can Add Shortcode in Page by Clicking  Alpharage Icon in Visual ToolBar and popup appear where you can manage your Testimonials Style Settings.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `alpharage-testimonials.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
